# IBEW Aptitude Test Prep — Study Website

A comprehensive, free study website for the **Electrical Training Alliance (IBEW/NJATC) Aptitude Test**. Built to help apprentices pass the exam and start their electrical career.

## 📚 What's Included

| Page | Content |
|------|---------|
| **Home** | Test overview, stats, navigation |
| **Algebra** | Full 12-section algebra curriculum — PEMDAS through number series — with **31 practice questions** |
| **Reading** | Reading strategies, 4 full passages with **20 comprehension questions** |
| **Practice Test** | Full-length simulated exam — **69 questions** (33 algebra + 36 reading) with timer and scoring |
| **Study Guide** | 8-week study schedule, formula cheat sheet, common mistakes, test-day tips |

## 📊 Test Structure

- **Algebra & Functions**: 33 questions, 46 minutes — no calculators allowed
- **Reading Comprehension**: 36 questions, 51 minutes
- **Scoring**: Stanine scale 1–9 (minimum 4/9 to qualify at most locals)
- **Retake Policy**: Wait 3 months between attempts

## 🚀 How to Use

1. Open `index.html` in any browser—no installation needed
2. Start with Algebra → work through each topic
3. Move to Reading → practice passage strategies
4. Take the Full Practice Test under timed conditions
5. Review weak areas using the Study Guide

## 🧠 Topics Covered

**Algebra**: PEMDAS · Evaluating expressions · Linear equations · Systems of equations · Polynomials · Factoring · FOIL · Functions & graphs · Slope-intercept · Number series · Exponents · Cartesian plane

**Reading**: Main idea · Specific details · Inference · Vocabulary in context · Author's tone · Timed strategies

## 🛠 Tech

Pure HTML + CSS + JavaScript. No frameworks, no build tools, no dependencies. Open `index.html` and go.
